# My Favorite Terminal Commands

| key binding                     | action                                    | notes |
|---------------------------------|-------------------------------------------|-------|
| `$ touch .gitignore && open $_` | create and open the file you just created |       |
| `$ pwd`                         | present working directory                 |       |
| `$ cd`                          | change directory                          |       |
| `$ ls -la`                      | list all                                  |       |
| `$ ..`                          | up one directory                          |       |
| `$ mv file1 file1`              | rename or move a file                     |       |
