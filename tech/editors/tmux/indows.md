# Windows

| binding       | action                 | note                                         |
|---------------|------------------------|----------------------------------------------|
| `Preview + x` | kill window            | Type "y" when it asks you to delete a window |
| `Preview + c` | new window             |                                              |
| `Preview + x` | move to next window    |                                              |
| `Preview + p` | move to previes window |                                              |
